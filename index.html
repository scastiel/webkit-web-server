Webkit-web-server

Simple webserver using node-webkit.

Download .zip Download .tar.gz View on GitHub

You need to install node-webkit first.

To build the application, simple zip all files to webkit-web-server.nw (for example) :

zip webkit-web-server.nw *

Then open the file with the node-webkit executable.
